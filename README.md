## 2026 每日时间轴（可在线部署）

这是一个**纯静态网页**（只有 `index.html`），手机/电脑浏览器都能打开。

### 功能

- **2026 年日历**：标注中国法定节假日与调休补班（来源：中国政府网通知）。
- **每日 24 小时时间轴**：每小时可选颜色、填写文字。
- **活动预设**：每小时可从“上课/睡觉/通勤/运动…”等预设一键填入。
- **按色系快捷选色**：蓝/绿/黄/橙/红/紫/灰等色板。
- **导出图片**：点击页面右上角 **「导出 PNG」**，下载当天时间轴图片。
- **自动保存**：填写内容会自动存到浏览器 `localStorage`；回到已填写的日期会自动恢复内容；日历上会用小圆点标记“已填写”的日期。

### 重要说明（数据保存范围）

- 目前的“保存”是 **同一台设备、同一浏览器** 内生效（比如你手机上的数据不会自动同步到电脑）。
- 如果你需要“跨设备同步”（同一个在线链接在手机/电脑都能看到同一份数据），需要加一个云端后端（例如 Firebase/Supabase/自建接口）。你说一声我可以继续改成账号/同步版。

### 如何部署成在线网页（推荐：GitHub Pages）

1. 在 GitHub 新建一个仓库（例如 `timeline-2026`）。
2. 把本目录的 `index.html`（以及可选的 `README.md`）上传到仓库根目录。
3. 打开仓库 **Settings → Pages**：
   - Build and deployment: 选择 **Deploy from a branch**
   - Branch: 选择 `main` / `master`，目录选 `/ (root)`
4. 保存后等待 1 分钟左右，就会得到一个在线地址。

### 其他平台（可选）

- **Vercel / Netlify**：把仓库导入即可（它们会自动识别静态站点并发布）。
